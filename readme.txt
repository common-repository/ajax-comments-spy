AJAX Comments Spy is a wordpress plugin which allows to add "LIVE Comments"
widget to the sidebar. The widget will check for approved comments every minute
(by default) and output the latest comments via the widget.

== Description ==

AJAX Comments Spy widget enables you to output comment updates to your users.

The plugin fetches the comments according to the settings in the
administration panel.

You can specify whether you want to spy pages, posts or both. You can change
the title of the widget, total amount of posts to output and update interval
time. You can configure to output comments from password protect posts.
